# Workshophy Service
This service is responsible for:
* Many things for many roles

## Tech stack
* Build tool: maven >= 3.9.5
* Java: 23
* Framework: Spring boot 3.2.x
* DBMS: MySQL

## Prerequisites
* Java SDK 23
* A MySQL server

## Start application
`mvn spring-boot:run`

## Build application
`mvn clean package`
